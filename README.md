# Nauka-Python
Dokumentacja progresu nauki języka programowania Python. Nazywam się Dariusz Kubicki i kilka dni temu dowiedziałem się o ogólnopolskim konkursie Gigathon. Postanowiłem, że spróbuję swoich sił i do niego podejdę. Pierwszy etap był dla mnie banalny, lecz dalsze etapy będą wymagały dobrej znajomości Pythona, dlatego zaczynam naukę tego języka, a wszystkie notatki i postępy będę tutaj dokumentował. Do końca drugiego etapu zostało mi 12 dni, więc odkładam całą naukę Arduino, C++ i Linuxa na później, a teraz skupię się w 100% na Pythonie.

## 📅 Plan przygotowań do Gigathonu (12 dni)
### 🟩 Etap 1: Fundamenty (Dni 1-3)
[ ] Składnia i typy danych: operacje na int, float, str oraz bool.

[ ] Struktury danych: biegłość w obsłudze list (listy), dict (słowniki) oraz set (zbiory).

[ ] Instrukcje warunkowe: if, elif, else oraz operatory logiczne (and, or, not).

[ ] Pętle: efektywne wykorzystanie for oraz while.

### 🟨 Etap 2: Logika i Algorytmy (Dni 4-8)
[ ] Funkcje: definiowanie def, przekazywanie argumentów i zwracanie wartości return.

[ ] List Comprehension: tworzenie list w jednej linii (bardzo przydatne na konkursach).

[ ] Wbudowane funkcje: znajomość min(), max(), sum(), sorted(), len().

[ ] Praca z tekstami: metody stringów (split, join, replace, slicing).

### 🟥 Etap 3: Trening Bojowy (Dni 9-12)
[ ] Biblioteki standardowe: math (pierwiastki, silnie), random oraz datetime.

[ ] Obsługa wejścia/wyjścia: szybkie wczytywanie danych (input) i formatowanie wyjścia (f-strings).

[ ] Rozwiązywanie zadań z poprzednich edycji: analiza problemów algorytmicznych.

[ ] Debugowanie: umiejętność szybkiego znajdowania błędów w kodzie.
